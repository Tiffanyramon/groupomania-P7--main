Technologie utilisées:

Backend: Node.js 
Database: MySQL



Packages: 

- bcrypt
- cors
- dotenv
- express
- helmet
- jsonwebtoken
- multer
- mysql
- nodemon
- password-validator
- path

Installation 

Ouvrir un terminal:

1. cd backend
2. npm install
3. npm install mysql --save
4. Configuration du fichier .env 
  Crée un fichier .env à la racine du projet du côté backend mettre la clé de chriffrement de: 
    - JWT_SECRET
    - DB_HOST
    - DB_USER 
    - DB_PASSWORD
    - DB_NAME
5. nodemon server

Exécuter le backend sur http://localhost:3001 seulement
